# Excel_Project
# 📊 Bike Buyers Sales Analysis Dashboard

An interactive Excel dashboard built to analyze customer purchasing behavior using Pivot Tables, Pivot Charts, and Slicers. This project demonstrates the complete data analysis workflow, from cleaning raw data to creating an interactive dashboard for business insights.

---

## 📌 Project Overview

The objective of this project was to transform a raw bike buyers dataset into an interactive dashboard that helps users explore customer demographics and purchasing patterns.

The project follows a structured data analysis process:

- Data Cleaning
- Data Transformation
- Pivot Table Analysis
- Data Visualization
- Interactive Dashboard Creation

## 📂 Project Workflow

### 1. Raw Dataset
Started with the original Bike Buyers dataset containing customer demographic and purchasing information.

📷 *Screenshot: Raw Data*

---

### 2. Data Cleaning & Preparation

Prepared the dataset for analysis by:

- Removing duplicate records
- Standardizing inconsistent values
- Creating Age Brackets (Adolescent, Middle Age, and Old)
- Organizing the dataset for Pivot Table analysis

📷 *Screenshot: Working Sheet / Cleaned Data*

---

### 3. Data Analysis Using Pivot Tables

Built Pivot Tables to summarize customer behavior and purchasing trends, including:

- Average Income by Gender
- Bike Purchase by Commute Distance
- Bike Purchase by Age Bracket
- Customer Purchase Distribution

📷 *Screenshot: Pivot Tables*

---

### 4. Interactive Dashboard

Designed an interactive dashboard using Pivot Charts and Slicers, allowing users to filter and analyze the data dynamically.

The dashboard enables quick analysis of:

- Customer Income
- Purchase Decisions
- Commute Distance
- Age Brackets
- Customer Demographics

📷 *Screenshot: Dashboard*

## 📈 Key Insights

Some insights obtained from the analysis include:

- Female customers have a higher average income than male customers.
- Middle-aged customers represent the largest customer segment.
- Commute distance influences customer purchasing behavior.
- Interactive filters allow users to explore the data from different perspectives.
